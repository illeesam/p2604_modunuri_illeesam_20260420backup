# EC 정책서 인덱스

전자상거래(EC) 도메인 비즈니스 정책 문서 모음.
도메인별 하위 폴더로 분리 관리.

## 도메인 폴더 구조

| 폴더 | 도메인 | 내용 |
|---|---|---|
| `정책서ec-mb/` | 회원 (MB) | 회원 상태·등급·SNS연동·탈퇴 |
| `정책서ec-pd/` | 상품 (PD) | 카테고리·상품·배송템플릿·묶음·세트·사은품 |
| `정책서ec-od/` | 주문 (OD) | 주문·결제·배송·클레임(취소/반품/교환/부분처리) |
| `정책서ec-pm/` | 프로모션 (PM) | 쿠폰·캐시·적립금·할인·사은품·상품권·기획전·이벤트 |
| `정책서ec-dp/` | 전시 (DP) | UI·Area·Panel·Widget·WidgetLib |
| `정책서ec-st/` | 정산 (ST) | 마감·지급·수집원장·대사·ERP전표 |

## 파일명 패턴
`{domain}.{NN}.{설명}.md`
- domain: mb / pd / od / pm / dp / st
- NN: 01부터 시작, 새 파일은 해당 도메인 마지막 번호 + 1

## 수정 규칙
- DDL 변경 시 관련 정책서 **반드시** 함께 수정
- 새 테이블/기능 추가 시 정책서도 신규 생성 후 해당 도메인 `CLAUDE.md` 파일 목록 업데이트
- 상태 코드 변경 시 `sy.08.공통코드.md`도 함께 수정
