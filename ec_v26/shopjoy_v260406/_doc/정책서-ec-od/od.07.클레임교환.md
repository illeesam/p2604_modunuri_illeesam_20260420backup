# 318. 클레임 - 교환 정책

## 목적
배송 받은 상품의 교환 요청 및 처리 정책 정의

## 범위
- 교환 신청 및 승인
- 반품 수거 및 교환상품 배송
- 교환 배송료 정책
- 교환 불가 상황 처리

## 교환 상태 (CLAIM_STATUS)
| 상태 | 코드 | 설명 |
|------|------|------|
| 요청 | REQUESTED | 교환 신청 |
| 승인 | APPROVED | 관리자 승인 |
| 수거중 | IN_PICKUP | 반품 수거 중 |
| 배송중 | IN_TRANSIT | 교환상품 배송 중 |
| 완료 | COMPLT | 교환 완료 |
| 거절 | REJECTED | 교환 불가 |
| 취소 | CANCELLED | 교환 철회 |

## 교환 사유 (CLAIM_REASON)
| 사유 | 코드 | 배송료부담 | 예상기간 |
|------|------|-----------|---------|
| 상품하자 | DEFECT | 판매자 100% | 7~10일 |
| 배송손상 | DAMAGE | 판매자 100% | 7~10일 |
| 오배송 | WRONG_ITEM | 판매자 100% | 7~10일 |
| 사이즈오류 | SIZE_ERROR | 판매자 100% | 5~7일 |
| 색상오류 | COLOR_ERROR | 판매자 100% | 5~7일 |
| 단순변심 | CHANGE_OF_MIND | 수거배송비 고객 50%/판매자 50%, 발송배송비 고객 100% | 7~10일 |
| 기타 | OTHER | 협의 | - |

## 주요 정책

### 1. 교환 신청 가능 기간
- **배송완료 후**: 30일 이내
- **자동폐기**: 30일 경과 시 자동거절
- **예외**:
  - 상품하자: 180일 이내 가능
  - 배송손상: 수령 후 7일 이내

### 2. 교환 프로세스
1. **신청**: 고객이 교환 신청
   - 기존상품 정보
   - 교환요청 상품 선택
   - 사유 선택 + 상세설명
2. **검토**: 담당자 검토
   - 사유 적정성 확인
   - 교환가능성 판단
   - 사이즈/색상 재고 확인
3. **승인**: 교환 승인
   - 수거택배사 선정
   - 교환상품 배송 준비
4. **반품수거**: 기존 상품 수거
   - 택배사 픽업
   - 3~5 영업일
5. **반품입고**: 기존 상품 입고
   - 검수 진행
   - 문제 있으면 재협의
6. **교환배송**: 교환상품 발송
   - 입고 확인 후 배송
   - 3~5 영업일 추가
7. **완료**: 고객 수령 및 확정

### 3. 교환 배송료 정책
- **왕복 배송료**:
  - 수거료 (고객 → 회사): 배송료 부담자 결정
  - 발송료 (회사 → 고객): 판매자 또는 고객
- **배송료 부담**:
  - 상품하자/배송손상/오배송: 판매자 100%
  - 사이즈/색상오류: 판매자 100%
  - 단순변심: 수거배송비 고객 50%/판매자 50%, 발송배송비 고객 100%
- **배송료 계산**:
  - 기본배송료 × 2 (수거 + 발송)
  - 도서산간: 실비 + 추가요금

### 4. 교환상품 선택
- **가격대변경**: 가능 (차액 정산)
  - 높은가격으로 교환: 고객이 추가비용 결제
  - 낮은가격으로 교환: 환불 또는 캐시 적립
- **재고부족**: 
  - 완전동일품목 재고부족 시 유사상품 추천
  - 고객이 거절하면 반품처리

### 5. 교환 제약사항
- **위생상품**: 교환 불가 (반품만 가능)
- **오염/손상**: 명확한 사용흔적은 교환 불가
- **수량초과**: 1회 교환 최대 5개

### 6. 교환비용 정산
- **배송료 공제**: 반품 입고 시
- **추가비용 결제**: 교환상품 배송 전
- **환불처리**: 교환상품이 저가 시 원금 환불 또는 캐시 적립

### 7. 교환실패 처리
- **배송 실패**: 반복 시도 (3회)
- **수령거절**: 자동반품 처리
- **불량상품**: 추가 교환 진행

### 8. 동일 주문 내 복수 아이템 동시 클레임
- **원칙**: `od_order_item`은 독립적으로 클레임 처리 → 같은 주문의 다른 아이템에 클레임이 진행 중이어도 별도 교환 신청 가능
- **예시**: 주문 1건에 5개 order_item → 3개 교환 진행 중 → 나머지 2개도 각각 교환 신청 가능
- **조건**: 각 order_item의 클레임 가능 상태(DELIVERED/CONFIRMED, 30일 이내) 충족 필요
- **제약**: 이미 클레임 진행 중인 order_item에는 중복 신청 불가
- **자동확정 타이머**: 교환 진행 중인 order_item은 DELIVERED 후 7일 자동확정 타이머에서 제외. 교환 완료(COMPLT) 또는 철회(CANCELLED) 시에만 타이머 재가동
- **order_status_cd 영향**: 일부 아이템 교환 진행 중에는 집계 상태 COMPLT 전환 보류

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| claim_id | 클레임ID | YYMMDDhhmmss+rand4 |
| claim_type_cd | 클레임유형 | EXCHANGE (고정) |
| claim_status_cd | 교환상태 | CLAIM_STATUS 코드 |
| claim_status_cd_before | 변경전상태 | 상태변경 추적 |
| return_status_cd | 수거상태 | DLIV_STATUS 코드 |
| return_shipping_fee | 수거배송료 | BIGINT |
| return_courier_cd | 수거택배사 | COURIER 코드 |
| exchange_shipping_fee | 발송배송료 | BIGINT |
| exchange_courier_cd | 발송택배사 | COURIER 코드 |
| exchange_tracking_no | 발송송장 | 택배사 송장번호 |
| outbound_dliv_id | 발송배송ID | od_dliv FK |
| inbound_dliv_id | 수거배송ID | od_dliv FK |
| total_shipping_fee | 총배송료 | BIGINT (수거+발송) |
| reason_cd | 교환사유 | CLAIM_REASON 코드 |
| reason_detail | 상세사유 | TEXT |

## 관련 테이블
- od_claim: 교환 요청
- od_claim_item: 교환 상품 명세
- od_dliv: 수거/발송 배송 정보 (2개)
- od_order_item: 교환상품 연결 (새주문)

## 제약사항
- 교환기한(30일) 경과 시 자동거절
- 반품입고 전까지 교환상품 배송 불가
- 교환상품 재고 부족 시 고객 동의 필수
