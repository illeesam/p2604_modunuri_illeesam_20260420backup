# 316. 클레임 - 취소 정책

## 목적
주문 취소 요청 처리 및 취소 환불 정책 정의

## 범위
- 취소 신청 및 승인
- 취소 환불 처리
- 취소 수수료 정책

## 취소 상태 (CLAIM_STATUS)
| 상태 | 코드 | 설명 |
|------|------|------|
| 요청 | REQUESTED | 취소 신청 |
| 승인 | APPROVED | 관리자 승인 |
| 거절 | REJECTED | 취소 불가 |
| 완료 | COMPLT | 취소 완료 |
| 취소 | CANCELLED | 취소 철회 |

## 취소 사유 (CLAIM_REASON)
| 사유 | 코드 | 특징 |
|------|------|------|
| 단순변심 | CHANGE_OF_MIND | 배송 전 |
| 상품하자 | DEFECT | 배송 후 |
| 배송지연 | DELAY | 약속기한 초과. 결제완료 후 배송준비 착수 전 단계에만 적용 가능 (배송 착수 후 취소 불가) |
| 오배송 | WRONG_ITEM | 상품 오류 |
| 기타 | OTHER | 기타사유 |

## 주요 정책

### 1. 취소 신청 가능 기간
- **결제전**: 제한 없음 (24시간 내 자동취소)
- **배송전**: 주문일로부터 7일 이내
- **배송중**: 불가능 (고객센터 문의)
- **배송후**: 불가능 (반품으로 처리)

### 2. 취소 유형별 처리

#### 2-1. 일반취소 (배송 전)
- **신청**: 고객 직접 신청 또는 고객센터
- **승인**: 자동 승인 (관리자 개입 안함)
- **환불**: 즉시 환불
- **환불방식**: 원 결제 수단으로
- **수수료**: 무료

#### 2-2. 사유취소 (배송 전)
- **신청**: 상품 하자, 오배송 등
- **승인**: 관리자 검토 (1~2일)
- **증거자료**: 사진 등 첨부
- **환불**: 즉시 환불
- **수수료**: 무료

### 3. 부분취소 (order_item 단위)
- **신청 기준**: order_item 단위로 취소 신청
- **환불금액**: 취소하는 order_item만 해당 주문금액 환불
  - 전체 주문금액 중 부분취소분만 환불
  - 무료배송 → 유료배송 전환 가능
  - 배송료 재계산
- **제약**:
  - 클레임 진행중인 order_item은 신청 불가
  - order_item당 클레임은 기본적으로 **1회만 가능**
  - **예외**: 교환 후 추가 취소 가능 (교환-취소 순서)

### 4. 동일 주문 내 복수 아이템 동시 클레임
- **원칙**: `od_order_item`은 독립적으로 클레임 처리 → 같은 주문의 다른 아이템에 클레임이 진행 중이어도 별도 클레임 신청 가능
- **예시**: 주문 1건에 5개 order_item → 3개 클레임 진행 중 → 나머지 2개도 각각 취소 신청 가능
- **조건**: 각 order_item의 클레임 가능 상태(ORDERED/PAID) 충족 필요
- **제약**: 이미 클레임 진행 중인 order_item에는 중복 신청 불가
- **order_status_cd 영향**: 모든 아이템이 취소/완료될 때까지 CANCELLED 전환 보류. 일부만 취소되면 나머지 아이템 상태로 집계 유지

### 4. 취소 환불 정책
- **환불기한**: 신청일로부터 3~5 영업일
- **환불방식**:
  - 카드: 카드사 통해 자동환불
  - 계좌: 등록 계좌로 송금
  - 휴대폰: 통신사 통해 환불
- **환불금액**: (주문금액 × 취소수량/주문수량) - 배송료 (배송 후 취소 시)

### 5. 취소 제약사항
- **배송시작후**: 취소 불가
  - 배송진행상황 실시간 조회
  - 운송중 상태면 취소 불가
- **특수상품**: 일부 상품 취소 불가
  - 디지털 상품: 구매즉시 제공되므로 취소 불가
  - 주문제작: 생산시작후 취소 불가
  - 식품: 냉동/신선 상품은 배송시작후 취소 불가
- **클레임 진행 제약**: 진행중인 order_item은 신규 클레임 불가
- **1회 원칙**: order_item당 1회만 클레임 가능 (취소/반품/교환 중 1가지)

### 6. 취소 수수료
- **기본정책**: 무수수료
- **배송 후 취소**: 왕복배송료 공제
  - 배송료는 취소 완료 후 환불금액에서 공제

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| claim_id | 클레임ID | YYMMDDhhmmss+rand4 |
| claim_type_cd | 클레임유형 | CANCEL (고정) |
| claim_status_cd | 취소상태 | CLAIM_STATUS 코드 |
| claim_status_cd_before | 변경전상태 | 상태변경 추적 |
| reason_cd | 취소사유 | CLAIM_REASON 코드 |
| reason_detail | 상세사유 | 300자 이내 |
| refund_method_cd | 환불수단 | REFUND_METHOD_CD 코드 |
| refund_amt | 환불금액 | BIGINT |
| request_date | 신청일시 | TIMESTAMP |
| proc_date | 처리일시 | TIMESTAMP |
| proc_user_id | 처리자 | sy_user.user_id |

## 관련 테이블
- od_claim: 취소 요청
- od_claim_item: 취소 상품 명세
- od_pay: 결제 정보 (환불 연결)
- od_claim_chg_hist: 취소 상태 변경 이력

## 제약사항
- 배송 시작 후 취소 불가
- 반품/교환 진행 중인 주문의 취소 불가
- 취소 후 재주문은 새로운 주문으로 처리
