# pm.02. 프로모션 - 쿠폰 정책

## 목적
쿠폰 발행, 배포, 사용을 통한 판매 촉진 정책 정의

## 범위
- 쿠폰 생성 및 발행
- 쿠폰 할당 및 배포
- 쿠폰 사용 및 환불
- 쿠폰 상태 관리
- 자사/판매자 부담금 분담 설정

## 쿠폰 상태 (COUPON_STATUS)
| 상태 | 코드 | 설명 |
|------|------|------|
| 활성 | ACTIVE | 사용 가능 상태 |
| 비활성 | INACTIVE | 사용 불가 상태 |
| 만료 | EXPIRED | 유효기간 만료 |

## 쿠폰 유형 (COUPON_TYPE)
| 유형 | 코드 | 설명 | 예시 |
|------|------|------|------|
| 정률 | RATE | 할인율 적용 | 20% 할인 |
| 정액 | FIXED | 고정금액 할인 | 10,000원 할인 |

## 쿠폰 대상 (COUPON_TARGET)
| 대상 | 코드 | 설명 |
|------|------|------|
| 전체 | ALL | 모든 회원 |
| 회원 | MEMBER | 특정 회원 그룹 |
| 등급 | GRADE | 특정 회원등급 (VIP, GOLD 등) |

## 주요 정책

### 1. 쿠폰 생성
- **필수정보**:
  - 쿠폰명 (100자 이내)
  - 쿠폰코드 (50자, UNIQUE)
  - 할인유형 (정률/정액)
  - 할인율/금액
  - 최소주문금액
- **선택정보**:
  - 최대할인한도
  - 발급한도 / 회원당 최대발급수
  - 유효기간
  - 최소주문수량
  - 적용 회원등급 제한

### 2. 쿠폰 발행 방식
- **일반발행**: 특정 기간 공개 발행
- **특정회원 배포**: 타겟 회원에게만 발급
  - 생일쿠폰, 등급별쿠폰, 휴면복구쿠폰 등
- **이벤트**: 이벤트 참여 시 쿠폰 지급
- **추천**: 친구 추천 보상 쿠폰

### 3. 자사/판매자 부담금 분담
쿠폰 할인액의 비용 부담 주체를 설정:
- `self_cdiv_rate`: 자사(사이트) 분담율 (%) — 기본 100%
- `seller_cdiv_rate`: 판매자(업체) 분담율 (%) — 기본 0%
- `seller_cdiv_remark`: 판매자 분담 비고 (특약 내용 등)
- **규칙**: `self_cdiv_rate + seller_cdiv_rate = 100`%
- 예시: 자사 70% + 판매자 30% → 쿠폰 할인액 10,000원 중 자사 7,000원, 판매자 3,000원 부담
- 정산 시 `seller_cdiv_rate` 비율만큼 판매자 정산액에서 차감

### 4. 채널별 적용 설정
쿠폰을 특정 채널에서만 사용 가능하도록 제한:
- `dvc_pc_yn`: PC 브라우저 채널 (Y=적용)
- `dvc_mweb_yn`: 모바일 웹 채널 (Y=적용)
- `dvc_mapp_yn`: 모바일 앱 채널 (Y=적용)
- 모두 Y이면 채널 제한 없음

### 5. 회원등급 제한
- `mem_grade_cd`: 특정 등급에만 쿠폰 적용 (NULL=전체 등급)
- 예: VIP 전용 쿠폰 → `mem_grade_cd = 'VIP'`
- `target_type_cd / target_value`와 병행 사용 가능

### 6. 쿠폰 적용 규칙
- **최소주문금액** (`min_order_amt`): 기본 조건 필수
- **최소주문수량** (`min_order_qty`): 수량 기준 조건 추가 (NULL=제한없음)
- **최대할인한도**: 정률쿠폰 시에만 적용
- **중복사용**: 1회 주문당 쿠폰 1개만 사용
- **선택적용**: 최적 할인을 자동으로 선택

### 7. 쿠폰 할인 한계
- **정률할인**: 최대 50% 이내
- **정액할인**: 최소 1,000원 ~ 최대 100,000원
- **중복할인**: 쿠폰 + 카테고리할인 + 상품할인은 1개만 적용

### 8. 쿠폰 유효기간
- **기본기간**: 3개월
- **장기쿠폰**: 최대 1년
- **자동만료**: 유효기간 종료 시 자동 사용불가
- **만료알림**: 7일 전 회원에게 알림

### 9. 쿠폰 발급한도
- **총발급한도** (`issue_limit`): 최대 10,000개 (NULL=무제한)
- **회원당한도** (`max_issue_per_mem`): 기본 3개까지 (NULL=무제한)
- **일일발급한도**: 최대 100개/일

### 10. 쿠폰 취소 및 환불
- **미사용 취소**: 자동 반납 (유효기간 만료)
- **사용후 취소**: 
  - 주문취소 시 쿠폰 자동 복구
  - 반품/교환 시 적립금으로 환불
- **쿠폰부정**: 중복사용, 불법양도 적발 시
  - 계정 정지 및 쿠폰 회수

| coupon_nm | 쿠폰명 | 100자 이내 필수 |
| coupon_type_cd | 할인유형 | RATE/FIXED |

| min_order_qty | 최소주문수량 | NULL=제한없음 |
| max_discount_amt | 최대할인한도 | BIGINT (정률용) |
| issue_limit | 총발급한도 | INTEGER NULL=무제한 |
| max_issue_per_mem | 회원당 최대발급수 | NULL=무제한 |
| coupon_status_cd | 상태 | COUPON_STATUS 코드 |
| coupon_status_cd_before | 변경전상태 | 상태변경 추적 |

| mem_grade_cd | 적용등급 | NULL=전체, MEMBER_GRADE 코드 |
| self_cdiv_rate | 자사분담율 | DECIMAL(5,2) 기본 100% |
| seller_cdiv_rate | 판매자분담율 | DECIMAL(5,2) 기본 0% |
| seller_cdiv_remark | 판매자분담비고 | VARCHAR(300) |
| dvc_pc_yn | PC채널 | Y/N 기본 Y |
| dvc_mweb_yn | 모바일WEB채널 | Y/N 기본 Y |
| dvc_mapp_yn | 모바일APP채널 | Y/N 기본 Y |

## 쿠폰 적용 대상 항목 (pm_coupon_item)
- `target_type_cd`: PRODUCT / CATEGORY / VENDOR / BRAND
- 항목이 등록되지 않으면 전체 상품에 적용 (pm_coupon.target_type_cd = ALL)
- 대상이 지정된 경우 해당 상품/카테고리/판매자/브랜드에 속한 상품에만 적용

## 관련 테이블
- `pm_coupon`: 쿠폰 마스터
- `pm_coupon_item`: 쿠폰 적용 대상 (상품/카테고리/판매자/브랜드)
- `pm_coupon_issue`: 회원-쿠폰 발급
- `pm_coupon_usage`: 쿠폰 사용 이력
- `od_order`: 쿠폰 사용내역 (coupon_id)

## 제약사항
- 정률쿠폰의 할인율은 최대 50% 초과 불가
- 중복 사용 방지: 회원당 쿠폰 사용 1회 주문당 1개
- 유효기간 만료 자동 처리
- `self_cdiv_rate + seller_cdiv_rate = 100`% 유지 (애플리케이션 레이어 검증)
- 상품에 `coupon_use_yn = N` 설정 시 해당 상품 포함 주문에 쿠폰 적용 불가
