# 513. 프로모션 - 적립금 정책

## 목적
구매 장려 및 고객 충성도 증진을 위한 적립금 정책

## 범위
- 적립금 지급
- 적립금 사용
- 적립금 유효기간
- 적립금 환급

## 주요 정책

### 1. 적립금 지급 기준
- **구매적립**: 결제완료 금액의 1%
  - 쿠폰/할인금은 제외 (실제 결제액 기준)
  - 배송료는 미포함
- **회원등급별**: 
  - BASIC(일반): 1%
  - SILVER(실버): 1.5%
  - GOLD(골드): 2%
  - VIP(VIP): 2.5%
- **추가적립**:
  - 리뷰작성: 상품가의 0.5%
  - 이벤트: 행사별 상이
  - 생일: 5,000포인트
  - 레벨업: 10,000포인트

### 2. 적립금 지급 시점
- **구매적립**: 배송완료 후 5일
  - 반품 없음 확인 후 지급
- **리뷰적립**: 리뷰 승인 후 즉시
- **이벤트적립**: 이벤트 종료 후 일괄 지급

### 3. 적립금 사용
- **사용단위**: 1포인트
- **사용상한**: 1회 결제액의 50% 이내
- **사용범위**: 
  - 상품 구매
  - 배송료
  - 부분결제 (다른 결제수단과 병행)
- **환산**: 1포인트 = 1원

### 4. 적립금 유효기간
- **적립기간**: 3년 (최종 적립일로부터)
- **자동소멸**: 유효기간 경과 시 자동 소멸
- **소멸알림**: 30일 전, 7일 전 알림
- **만료연장**: 새로운 구매 또는 적립 시 자동 연장

### 5. 적립금 사용 취소
- **주문취소**: 적립금 반환
- **반품처리**: 
  - 부분반품: 해당 적립금만 반환
  - 전체반품: 전액 반환
- **환불방식**: 원 적립금 형태로 복구

### 6. 적립금 환급
- **환금불가**: 적립금은 현금 환전 불가
- **전환가능**: 캐시로 전환 가능 (1:1)
  - 전환 시 10% 수수료 차감
- **대체방법**: 다음 구매에 사용

### 7. 적립금 관리
- **마이페이지**: 현황 및 사용내역 조회
- **추적**: 월별, 카테고리별 적립 현황
- **경고**: 소멸 예정 적립금 알림

## 주요 필드
| 필드 | 설명 | 규칙 |
|------|------|------|
| member_id | 회원ID | FK mb_member |
| accumulation_balance | 적립금잔액 | INTEGER 기본값 0 |
| accumulation_points | 누적적립금 | INTEGER |
| used_points | 사용적립금 | INTEGER |
| expire_date | 만료예정일 | TIMESTAMP |

## 적립금 계산 예시
```
결제금액: 100,000원
쿠폰할인: 10,000원
배송료: 2,500원
실결제액: 92,500원

적립금 (BASIC): 92,500 × 1% = 925포인트
적립금 (GOLD): 92,500 × 2% = 1,850포인트
```

## 적립금 테이블 구조
- `pm_save`: 통합 원장 (EARN/USE/EXPIRE/CANCEL/ADMIN 전체 이력)
- `pm_save_issue`: 지급 상세 (구매/이벤트/리뷰/추천/관리자별 발생 건, PENDING→CONFIRMED 상태 관리)
  - `save_issue_type_cd`: ORDER / EVENT / REVIEW / REFERRAL / ADMIN
  - `order_id`, `order_item_id`, `prod_id`: 구매적립 시 상품 추적
  - `expire_date`: 건별 소멸예정일
- `pm_save_usage`: 사용 상세 (주문 시 사용 건, order_id + order_item_id + prod_id)

## 관련 테이블
| 테이블명 | 한글설명 |
|---------|---------|
| `pm_save` | 적립금 통합 원장 (모든 입출 이력) |
| `pm_save_issue` | 적립금 지급 이력 (발생 건별 상태 추적) |
| `pm_save_usage` | 적립금 사용 이력 (주문 시 사용 건별) |
| `mb_member` | 회원 마스터 (적립금 잔액 요약 저장) |

## 제약사항
- 환금 불가능 (캐시로만 전환, 10% 수수료)
- 사용한도: 결제액의 50% 이내
- 유효기간: 최종 적립일로부터 3년
