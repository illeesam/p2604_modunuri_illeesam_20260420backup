# ddlPgsql-zz.기타/ 기타 DDL 참고 자료

## 파일 목록
- `zz.사용자 (구매자, 판매-관리자, 배송업체-관리자).txt` — 사용자 유형 설계 메모
- `zz.사용자 사이트.txt` — 사이트별 사용자 분리 정책 메모
- `zz.사용자에 카카오,네이버, 구글 ci 값 저장필요할까.txt` — 소셜 로그인 CI 저장 여부 검토
- `zz.상품 컬러를 상품 옵션으로 변경.txt` — 컬러 → 옵션 구조 전환 메모
- `zz.업체정보 (판매업체, 배송업체).txt` — 업체 유형 설계 메모
- `zz.이벤트관련필요.txt` — 이벤트 도메인 추가 요건 메모
- `zz.전시관련필요.txt` — 전시(Display) 도메인 추가 요건 메모
- `zz.첨부관련필요.txt` — 첨부파일 도메인 추가 요건 메모
- `zz.카테고리.txt` — 카테고리 구조 설계 메모
- `zz.쿠폰적용상품 필요.txt` — 쿠폰-상품 연결 요건 메모
- `add_code_comments.js` — DDL 파일에 코드 주석 자동 추가 스크립트

## 목적
DDL 설계 과정의 검토 메모, 결정 미확정 이슈, 보조 스크립트 모음.
`zz.` 접두사 파일은 확정 전 논의 사항으로, 실제 DDL 반영 여부 확인 필요.
