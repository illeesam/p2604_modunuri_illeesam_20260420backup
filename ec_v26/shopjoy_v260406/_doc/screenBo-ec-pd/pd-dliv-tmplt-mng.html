배송템플릿 관리

📦 배송템플릿 역할 (pd_dliv_tmplt)

  • 상품(pd_prod)에 배송방법·배송비·반품지·반품비·교환비를 일괄 지정하는 마스터 설정
  • 업체(vendor)별로 여러 템플릿 생성 가능 — 예: 기본택배 / 착불배송 / 직배송 등
  • 주문 시 상품의 dliv_tmplt_id 기준으로 배송비 자동 계산
  • 클레임(반품) 시 return_cost(편도), 클레임(교환) 시 exchange_cost(왕복) 적용
  • 반품지 주소(return_addr*)는 반품 수거 기본주소로 클레임 생성 시 자동 입력
  • base_dliv_yn = Y인 템플릿이 해당 사이트·업체의 기본 배송 설정으로 적용

배송템플릿 등록 / 수정
① 기본 설정
직배송·방문수령 선택 시 택배사 미입력
② 배송비 설정
착불 선택 시 0 입력 권장
예: 50000 → 5만원 이상 주문 시 배송비 무료
무료배송 조건을 만족해도 도서산간 추가비 별도 부과
③ 반품/교환 비용 설정
고객 귀책 반품 시 회원 부담 (편도)
수거비 + 재발송비 = 왕복 합산 금액
반품 수거 시 사용할 택배사 (발송 택배사와 다를 수 있음)
④ 반품지 주소
⚠️ 반품지 주소는 클레임(반품/교환) 처리 시 수거지 주소로 자동 입력됩니다. 정확한 주소를 입력하세요.
return_addr_zip · return_addr · return_addr_detail

💡 배송비 계산 시나리오 (pd_dliv_tmplt 필드 적용 순서)

  1. 주문금액 < free_dliv_min_amt  →  dliv_cost 부과
  2. 주문금액 ≥ free_dliv_min_amt  →  배송비 무료
       단, 도서산간 배송지 = 추가로 island_extra_cost 부과
  3. 반품 (고객 귀책): return_cost(편도)만큼 회원 부담
  4. 교환 (고객 귀책): exchange_cost(왕복) 회원 부담
       = 수거비(편도) + 재발송비(편도) 합산
  5. 판매자 귀책 (불량품·오배송): 반품·교환비 전액 판매자 부담 → 회원 청구 없음
