위젯 라이브러리 (dp_widget_lib) — 재사용 템플릿
dp_widget_lib: 여러 dp_widget 인스턴스가 공유하는 재사용 가능한 위젯 정의. template_html + config_schema 포함. is_system=Y 시 시스템 기본 제공.
No 위젯코드 (widget_code) 라이브러리명 (widget_nm) 유형 (widget_type_cd) 표시경로 (disp_path) 시스템 기본 사용여부 참조위젯수 액션
1 STD_IMAGE_BANNER 표준 이미지 배너 image_banner GLOBAL Y — 시스템 Y 12개
2 STD_PROD_SLIDER 표준 상품 슬라이더 product_slider GLOBAL Y — 시스템 Y 8개
3 CUSTOM_VIP_BANNER VIP 전용 배너 커스텀 event_banner FRONT.VIP존 N Y 3개
